Collective Memory Experiment

Learn About Humans.

This page talks to a MemPalace-backed agent running locally with uv run agent.py --data=humans/. Every conversation can become memory. The more people talk to it, the more the palace learns what humans care about, repeat, fear, build, avoid, and return to.

Verbatim by default Conversations are stored as exact words, not reduced to summary slogans.
Shared accumulation Point everyone at the same humans/ data directory and the memory grows together.
Local first The page can speak to your own agent endpoint on your own machine or network.
Or start own API:
uv run agent.py --data=humans/

Then chat. If multiple users use the same agent instance and data path, MemPalace keeps collecting the topics and language patterns they bring into the room.
